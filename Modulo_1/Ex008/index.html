Marcar um bloco com uma tag

Basta apertar ctrl+shift+p, na barra de pesquisa Wrap wifh Abbreviation, e em seguida a tag desejada (apenas a letra e não esqueça de confirmar com ENTER)

Texto Inserido / Sublinhado

Podemos marcar um texto como inserido para dar uma ênfase e indicar.

Existe também o sublinhado com a tag U (não semantica)

EVITAR

Evite usar o B para negrito, I para itálico e o U para sublinhado

USE o STRONG para negrito, o EM para itálico e o INS para o sublinhado..
